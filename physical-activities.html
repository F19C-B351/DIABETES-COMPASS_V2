🏃 Running with Diabetes

Essential tips and hints for safe running when managing diabetes. Follow these guidelines to stay active and healthy.

📊 Check Blood Sugar Before Running

  • ✓ Measure glucose 15–30 minutes before running
  • ✓ Safe starting range: 100–180 mg/dL (5.5–10 mmol/L)
  • ⚠️ Avoid if too low (<90 mg/dL) or too high (>250 mg/dL with ketones)

🍬 Always Carry Fast Sugar

  • ✓ Glucose tablets
  • ✓ Fruit juice or sports drink
  • ✓ Honey packets or candy
  • ✓ Recommended: 15–20g fast carbs

🆔 Wear Medical Identification

  • ✓ Medical ID bracelet or card
  • ✓ Include diabetes type
  • ✓ Emergency contact info
  • ✓ Medication/insulin details

⏱️ Monitor After Running

  • ✓ Check immediately after running
  • ✓ Check again 2–4 hours later
  • ⚠️ Lows can occur hours after exercise

🚶 Start Slowly

  • ✓ Begin with walking + jogging intervals
  • ✓ Short distances (10–20 minutes)
  • ✓ Gradually increase intensity

📈 Know How Running Affects Glucose

  • ↓ Usually lowers glucose (long steady runs)
  • ↑ Sometimes raises glucose (sprinting/intense intervals)
  • ✓ Keep a record of distance, duration, glucose

🍌 Pre-Run Snacks

Eat 30–60 minutes before running if glucose is low-normal:

Banana Yogurt Whole-grain toast Oatmeal

💧 Stay Hydrated

Dehydration can increase blood sugar:

Before running During long runs After running

⚠️ Watch for Low Blood Sugar Symptoms

Stop running immediately if you experience:

• Shaking • Sweating • Dizziness • Confusion • Fast heartbeat • Weakness

Test – Carry Sugar – Stay Safe
